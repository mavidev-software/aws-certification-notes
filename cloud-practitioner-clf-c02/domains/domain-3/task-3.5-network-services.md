### Security Groups vs NACLs Karşılaştırma

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                    DETAYLI KARŞILAŞTIRMA                                    │
├────────────────────────┬────────────────────────┬───────────────────────────┤
│        Özellik         │    Security Group      │          NACL             │
├────────────────────────┼────────────────────────┼───────────────────────────┤
│ Seviye                 │ Instance (ENI)         │ Subnet                    │
├────────────────────────┼────────────────────────┼───────────────────────────┤
│ State                  │ STATEFUL               │ STATELESS                 │
│                        │ (Return traffic auto)  │ (Her iki yön kural gerek) │
├────────────────────────┼────────────────────────┼───────────────────────────┤
│ Varsayılan             │ Tüm giden ✓           │ Tüm trafik ✓             │
│                        │ Tüm gelen ✗           │ (Varsayılan NACL)         │
├────────────────────────┼────────────────────────┼───────────────────────────┤
│ Kurallar               │ Sadece ALLOW           │ ALLOW ve DENY             │
├────────────────────────┼────────────────────────┼───────────────────────────┤
│ Kural Değerlendirme    │ Tüm kurallar           │ Numara sırasına göre      │
│                        │ değerlendirilir        │ (küçükten büyüğe)         │
├────────────────────────┼────────────────────────┼───────────────────────────┤
│ Instance başına        │ 5 SG'ye kadar          │ 1 NACL (subnet başına)    │
├────────────────────────┼────────────────────────┼───────────────────────────┤
│ Etki alanı             │ Sadece atandığı inst.  │ Tüm subnet kaynakları     │
└────────────────────────┴────────────────────────┴───────────────────────────┘

┌─────────────────────────────────────────────────────────────────┐
│                    KRİTİK SINAV NOKTASI!                        │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  Security Group = STATEFUL                                      │
│  ──────────────────────────                                    │
│  • Gelen trafiğe izin verirsen, cevap otomatik gidebilir       │
│  • Outbound kuralı GEREKMEZ                                    │
│                                                                 │
│  NACL = STATELESS                                              │
│  ──────────────                                                │
│  • Gelen ve giden trafik için AYRI kurallar gerekir           │
│  • Ephemeral ports (1024-65535) için outbound rule gerekli    │
│                                                                 │
│  Hangisini kullan?                                             │
│  • Security Group = İlk tercih, her zaman kullan              │
│  • NACL = Ek güvenlik katmanı, IP engelleme için              │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```
