## Önemli Konular

### Design Secure Architectures
- [ ] VPC tasarımı (subnets, NACLs, Security Groups)
- [ ] IAM best practices
- [ ] AWS Organizations ve SCP
- [ ] Encryption at rest ve in transit
- [ ] AWS Shield, WAF, GuardDuty

### Design Resilient Architectures
- [ ] Multi-AZ ve Multi-Region
- [ ] Auto Scaling stratejileri
- [ ] Load Balancing (ALB, NLB, GLB)
- [ ] Disaster Recovery stratejileri
- [ ] Backup ve restore

### Design High-Performing Architectures
- [ ] Compute seçimi (EC2, Lambda, ECS, EKS)
- [ ] Storage seçimi (S3, EBS, EFS, FSx)
- [ ] Database seçimi (RDS, Aurora, DynamoDB, ElastiCache)
- [ ] Caching stratejileri (CloudFront, ElastiCache)
- [ ] Network optimizasyonu

### Design Cost-Optimized Architectures
- [ ] EC2 pricing (On-Demand, Reserved, Spot, Savings Plans)
- [ ] S3 storage classes
- [ ] Right-sizing
- [ ] Cost monitoring (Cost Explorer, Budgets)
- [ ] Serverless vs traditional
